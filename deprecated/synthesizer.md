# synthesizer Wiki

## 结构规划模板库

### 标准幻灯片结构模板

**三段式（适用大部分场景）**
1. 标题页：主题 + 主讲人 + 日期
2. 内容页（3-5个章节）
   - 每章：章节标题页 + 2-3 内容页
3. 总结页：核心要点回顾 + 行动号召

**时间线式（适用历程/发展类主题）**
1. 标题页
2. 起点 → 关键节点1 → 关键节点2 → 当前状态 → 未来规划
3. 总结与展望

**对比式（适用选型/方案比较）**
1. 标题页
2. 背景与挑战
3. 方案A分析
4. 方案B分析
5. 对比总结
6. 推荐结论

**问题-解决式（适用技术分享）**
1. 标题页
2. 问题描述
3. 现有方案及其局限
4. 新方案介绍
5. 效果数据
6. 结论与展望

### 空间矩阵规划模板

**简单场景（无纵向嵌套）**
- 所有页面横向排列
- 用章节标题页分隔各部分

**中等复杂度（纵向嵌套子页面）**
```
<section>                    ← 第1章（横向）
  <section>章节标题</section> ← 纵向第1页
  <section>内容页1</section>  ← 纵向第2页
</section>
<section>                  ← 第2章（横向）
  <section>内容页2</section>
</section>
```

**复杂场景（多级嵌套）**
- 最多支持 2 级嵌套（横向→纵向）
- 不建议更多层级，用户体验差

### Fragment 规划规则

1. **单页最多 6 个 fragment**，超过则拆页
2. **顺序展示**：默认从上到下 / 从左到右
3. **分组展示**：同类内容放同一页
4. **数据页**：数字/图表高亮用 fragment

### 页面类型清单

| 页面类型 | 特点 | fragment 使用 |
|---------|------|--------------|
| 标题页 | 大标题 + 副标题 | 无 |
| 章节页 | 简洁标题 | 无 |
| 内容页 | 文字 + 图表 | 可选 |
| 要点页 | 列表形式 | 常用 |
| 引用页 | 大字体引言 | 无 |
| 数据页 | 图表 + 解读 | 可选 |
| 总结页 | 核心要点 | fragment 快速回顾 |
